# Retail Sales Data Analysis

## Objective
Analyze retail sales data to find sales trends, top products, and business insights.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Steps Performed
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Visualization of sales, price, category, and trends

## Results
- Identified top selling products
- Found sales distribution and category-wise performance
- Generated visual reports for business decisions
